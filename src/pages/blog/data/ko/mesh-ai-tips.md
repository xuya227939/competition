## 프롬프트 작성 기법

### 1. 구조화된 설명 방법

**기본 구조:**

**예시 비교:**

- ❌ 나쁜 설명: "의자"
- ✅ 좋은 설명: "금속 프레임과 조절 가능한 팔걸이가 있는 현대적인 스타일의 검은색 가죽 사무실 의자"

### 2. 키워드 사용 전략

**형태 키워드:**

- 기본 형태: 큐브, 원통, 구, 원뿔
- 복합 형태: L자형, U자형, 링 모양, 나선형
- 유기적 형태: 유선형, 곡선형, 불규칙형

**재질 키워드:**

- 금속: 스테인리스 스틸, 구리, 철, 알루미늄, 금, 은
- 목재: 참나무, 소나무, 호두나무, 체리나무, 대나무
- 석재: 대리석, 화강암, 석회암, 사암
- 직물: 면, 실크, 가죽, 펠트, 캔버스

**스타일 키워드:**

- 시대 스타일: 현대, 고전, 중세, 미래형
- 예술 스타일: 미니멀리스트, 바로크, 산업, 자연
- 문화 스타일: 북유럽, 일본, 미국, 유럽

### 3. 디테일 설명 기법

**크기 설명:**

- 구체적인 값 사용: "높이 30cm, 너비 50cm"
- 상대적 설명 사용: "작은, 중간, 큰"
- 비율 설명 사용: "가로 세로 비율 2:1"

**질감 설명:**

- 표면 질감: "매끄러운, 거친, 무광, 광택"
- 패턴 질감: "줄무늬, 체크무늬, 점무늬, 기하학적 패턴"
- 색상 그라데이션: "어두운 것에서 밝은 것으로, 그라데이션 효과"

**기능 설명:**

- 용도: "사무실 용, 가정용, 장식용"
- 특수 기능: "조절 가능, 접이식, 회전 가능"

## 매개변수 조정 최적화

### 1. 품질 매개변수 설정

**생성 품질 레벨:**

- **표준 품질**: 신속한 미리보기 및 컨셉 검증에 적합
- **고품질**: 대부분의 적용 시나리오에 적합
- **초고품질**: 전문 사용 및 최종 출력에 적합

**권장 설정:**

- 컨셉 단계: 신속한 반복을 위해 표준 품질 사용
- 개발 단계: 상세 디자인을 위해 고품질 사용
- 최종 단계: 완성품을 위해 초고품질 사용

### 2. 스타일 매개변수 조정

**스타일 강도:**

- **낮은 강도**: 원래 특성 유지, 약간의 스타일화
- **중간 강도**: 스타일과 실용성 균형
- **높은 강도**: 강한 스타일화, 두드러진 예술성

**스타일 일관성:**

- 전체 스타일 통일성 보장
- 스타일 충돌 방지
- 브랜드 톤 유지

### 3. 복잡성 제어

**지오메트리 복잡성:**

- **단순**: 기본 지오메트리, 신속한 프로토타이핑에 적합
- **중간**: 적당한 디테일, 대부분의 응용 프로그램에 적합
- **복잡**: 풍부한 디테일, 고품질 출력에 적합

**재질 복잡성:**

- 용도에 따라 적절한 재질 복잡성 선택
- 시각 효과와 성능 요구 균형
- 렌더링 및 내보내기 효율성 고려

## 입력 최적화 전략

### 1. 참조 이미지 사용

**이미지 선택 원칙:**

- 고품질의 선명한 참조 이미지 선택
- 이미지가 대상 모델과 관련되도록 보장
- 지나치게 복잡한 배경 간섭 방지

**이미지 전처리:**

- 이미지 크기 및 비율 조정
- 대비 및 선명도 향상
- 불필요한 배경 요소 제거

### 2. 다각도 참조

**시점 선택:**

- 정면도: 주요 특성 표시
- 측면도: 실루엣 및 비율 표시
- 평면도: 전체 레이아웃 표시

**참조 결합:**

- 여러 각도의 참조 이미지 결합
- 완전한 시각 정보 제공
- 생성 정확도 향상

### 3. 텍스트 설명 최적화

**설명 길이 제어:**

- 지나치게 긴 설명 방지(50-200단어 권장)
- 주요 특성 강조
- 간결하고 명확한 언어 사용

**설명 논리:**

- 중요도에 따라 설명 정렬
- 설명 논리 유지
- 모순되는 정보 방지

## 후처리 최적화 팁

### 1. 모델 최적화

**토폴로지 최적화:**

- 토폴로지 오류 확인 및 수정
- 폴리곤 분포 최적화
- 불필요한 정점 감소

**지오메트리 수리:**

- 비-매니폴드 엣지 수정
- 구멍 채우기
- 표면 매끄럽게 하기

### 2. 재질 조정

**재질 속성 최적화:**

- 재질 매개변수 조정
- 텍스처 매핑 최적화
- 조명 효과 개선

**색상 조정:**

- 색상 채도 조정
- 대비 최적화
- 색상 스타일 통일

### 3. 디테일 향상

**디테일 보완:**

- 필요한 디테일 요소 추가
- 엣지 처리 최적화
- 표면 질감 향상

**품질 검사:**

- 모델 무결성 확인
- 지오메트리 정확성 검증
- 렌더링 효과 테스트

## 일반적인 문제 해결

### 1. 생성 품질 문제

**문제 진단:**

- 프롬프트 설명이 명확한지 확인
- 매개변수 설정이 적절한지 확인
- 입력 이미지 품질 검증

**해결책:**

- 프롬프트 설명 최적화
- 생성 매개변수 조정
- 고품질 참조 이미지로 교체

### 2. 스타일 불일치

**문제 원인:**

- 프롬프트 설명이 충분히 구체적이지 않음
- 스타일 매개변수 설정이 부적절함
- 참조 이미지 스타일 충돌

**해결책:**

- 스타일 요구사항 명확화
- 스타일 매개변수 통일
- 일관된 참조 이미지 선택

### 3. 디테일 누락

**문제 분석:**

- 복잡성 설정이 너무 낮음
- 설명이 충분히 상세하지 않음
- 참조 이미지 정보 불충분

**개선 조치:**

- 복잡성 설정 증가
- 디테일 설명 추가
- 더 많은 참조 정보 제공
